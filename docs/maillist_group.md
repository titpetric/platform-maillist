# Maillist Group

This is the top level mail list group.

| Name       | Type         | Key | Comment                          |
|------------|--------------|-----|----------------------------------|
| id         | char(26)     | PRI | The ID assigned to the mail list |
| title      | varchar(255) |     | The name of the mail list        |
| created_at | timestamp    |     | Created At Time                  |
